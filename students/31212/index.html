IT 용어 사전

HTML
웹 페이지의 구조를 만드는 마크업 언어입니다.
CSS
웹 페이지의 디자인과 스타일을 지정하는 언어입니다.
JavaScript
웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다.
CPU
컴퓨터의 두뇌 역할을 하는 중앙처리장치입니다.
IP 주소
인터넷상에서 장치를 식별하는 고유한 주소입니다.
클라우드
인터넷을 통해 서버, 스토리지, 서비스를 제공하는 기술입니다.
URL
웹사이트의 주소를 나타내는 문자열입니다.
DNS
도메인 이름을 IP 주소로 바꿔주는 시스템입니다.
RAM
컴퓨터에서 데이터를 일시적으로 저장하는 빠른 메모리입니다.
SSD
기존 하드디스크보다 빠르고 조용한 저장장치입니다.
OS (운영체제)
컴퓨터를 작동시키는 기본 소프트웨어입니다. (예: Windows, macOS)
API
프로그램들이 서로 소통할 수 있게 하는 인터페이스입니다.
Algorithm
문제를 해결하기 위한 절차나 규칙입니다.
Git
코드의 변경 이력을 관리하는 도구입니다.
Open Source
누구나 자유롭게 사용하고 수정할 수 있는 소프트웨어입니다.
IoT (사물인터넷)
인터넷에 연결된 기기들이 데이터를 주고받는 기술입니다.
빅데이터
매우 크고 복잡한 데이터 집합을 의미합니다.
해킹
컴퓨터 시스템에 무단으로 침입하거나 조작하는 행위입니다.
2FA (이중 인증)
계정을 보호하기 위해 두 가지 인증 수단을 사용하는 방법입니다.
프론트엔드
사용자가 직접 보는 웹 화면을 구성하는 기술입니다.
백엔드
데이터 처리와 서버 기능을 담당하는 기술입니다.
클라이언트
서버에 요청을 보내는 사용자 쪽 프로그램입니다.
서버
클라이언트에게 정보를 제공하는 컴퓨터입니다.
