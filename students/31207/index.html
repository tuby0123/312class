🔐 해시 함수 체험 페이지 (SHA-256 시각화 + 해시테이블)

📘 해시 함수란?

해시 함수는 아무 길이의 입력 데이터를 고정된 길이의 '해시값'으로 바꿔주는 수학적인 변환 도구입니다.

  • 같은 입력은 항상 같은 해시값을 만들어냅니다.
  • 해시값만 보고는 원래 값을 알 수 없습니다 (단방향).
  • 입력을 아주 조금만 바꿔도 완전히 다른 해시값이 나옵니다.

💡 해시 함수는 비밀번호 저장, 파일 위변조 방지, 블록체인 등에서 사용됩니다.

🔐 해시값으로 로그인 시도하기

저장된 해시값(1234의 해시값)과 입력한 해시값이 같으면 로그인 성공!

예시:
입력: 1234
해시값: 03ac674216f3e15c761ee1a5e255f067953623c8f9664f0f4c9b5f9f2ed7a34d

📚 해시테이블 동작 원리

  1. 해시 함수: 키를 숫자(인덱스)로 변환해 저장 위치를 결정합니다.
  2. 데이터 저장: 변환된 인덱스 위치에 데이터를 저장합니다.
  3. 데이터 검색: 같은 해시 함수로 인덱스를 찾아 바로 데이터에 접근합니다.
  4. 충돌 문제: 서로 다른 키가 같은 인덱스로 변환될 수 있습니다.
  5. 충돌 해결(체이닝): 같은 인덱스에 여러 데이터를 연결 리스트로 저장합니다.
[0] -> (빈칸) [1] -> (빈칸) [2] -> (banana) [3] -> (cat) [4] -> (dog) [5] -> (apple) -> (leppa) // 충돌로 연결 리스트 형태 [6] -> (빈칸)
