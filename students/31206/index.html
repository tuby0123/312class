게임 코드 설명

자동차의 모든 움직임은 CarController.cs 스크립트로 제어됩니다. 주요 기능은 다음과 같습니다:

  • WheelControlInfo 구조체: 구동력, 제동력, 조향각도, 바퀴 설정 포함
  • StartEngine / StopEngine: 시동 켜기/끄기 처리 및 오디오 연동
  • HandleMotor: 속도에 따라 가속 토크를 조절하여 최고속도 제한
  • HandleSteering: 좌우 입력에 따라 앞바퀴의 조향각 조절
  • HandleBrake: 브레이크 입력에 따라 제동력 적용
  • UpdateWheelVisuals: WheelCollider의 위치/회전을 모델에 반영

🚗 예시 코드: 시동 기능


IEnumerator StartEngine()
{
    if (isStarting) yield break;
    isStarting = true;

    if (carAudioController != null)
        yield return StartCoroutine(carAudioController.StartEngine());
    else
        yield return null;

    isEngineOn = true;
    isStarting = false;
}
